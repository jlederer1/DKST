class dkst.dkst_datasets.DKSTDataset02(config_path)[source]

Bases: Dataset

A simple dataset of knowledge structures, conditional probabilities and responses data aggregates for training K-net. Loads or samples knowledge structures and response data according to given dataset configuration file. Retrieves a quadruple of 1D tensors, each of length 2^m, where m is the domain size.

Parameters:

  • config_path (str) – Path to the dataset configuration file.

  • baseset_path (str) – Path to the baseset file (optional).

Returns:

A dataset of knowledge structures and response aggregates.
